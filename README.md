# react-post-comment-v2
 
 ## Quick Start
 
 #### Install Package
  `npm install` 
 #### Start App
  `npm start`  
  
  Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

  #### Start json-server
  `npm run server`
  
Runs the app in the development mode.\
Open [http://localhost:3001](http://localhost:3001) to view it in the browser.
